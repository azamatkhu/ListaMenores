# Ejercicio ListaMenores

**Implementa un método que reciba una lista de objetos 
Persona y elimine todas las personas menores de 18 años usando Iterator.**

Para esto, voy a crear un clase llamado Persona con los atributos String nombre y int edad.

Despues, en MainListaMenores, meto en un ArrayList de Persona las personas aleatorias. Tambien, pongo los getters.
Con un iterador, compruebo si la persona es menor de 18 años (sacando su atributos con un getter .getEdad()), lo elimina.
